GigeCam Class

Revision: - Author: giulio






Device Properties
Property name
Property type
Description
Configuration Array of string Table of configurations for the cameras that could be connected by this device. Format is: label:ip address:offset_pos:gain_pos:img depth:exposure:gain:binning:updown:mirror:trigger mode:pixelXmm hor:pixelXmm ver: hor mech offset: ver mech offset For example: pil_camera_p1.1:192.168.1.10:10:0.01:8:10:350:1:0:0:1:15:10:0:0
AutoRoi Tango::DEV_BOOLEAN Enable autoroi (to minimize the area used for image processing
AutoRoiParam Array of double Parameters for autoroi configuration algorithm. param[0]=image decimation size (10 -> max image size) param[1]=low pass filter (1-10) param[2]=intensity threshold (0-10)
ImageProcessing Tango::DEV_BOOLEAN Enable image processing
PlotMode Array of short Set plot mode: short[0]= 0x1=plot axis & roi 0x2=plot spot contour 0x3=plot axis and spot contour short[1]= plot color
FastCalculation Tango::DEV_BOOLEAN Enable fast calculation algorithm.
ImageBufferSize Tango::DEV_SHORT Number of images stored in the circular buffer
DataBufferSize Tango::DEV_LONG Number of samples stored in the circular buffer for main parameters (horpos,verpos
SearchBackgroundParam Array of double Search background parameters: double[0]=number of background levels double[1]=derivative factor
RnmEnabled Tango::DEV_BOOLEAN Enable rnm data support
AutoConnect Tango::DEV_BOOLEAN If true connect to a camera when the device starts. (There must be only one camera supported).
RnmIndex Tango::DEV_LONG Rnm index of camera
ProcessMode Tango::DEV_LONG 0=raw 1=gaussian fit 2=gaussian asymmetric 4=confiteor
